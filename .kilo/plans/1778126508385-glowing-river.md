## Existing Pattern Audit
- **Mastra Framework:** Uses workflows with createStep, agents with generate method, tools with createTool. Current implementation has API usage errors.
- **Database:** Drizzle ORM with estimate_requests table (status: uploaded/processing/completed/failed).
- **UI:** RecentEstimatesWidget displays status badges correctly.
- **Status Flow:** uploaded (initial) -> processing (workflow start) -> completed/failed.
- **Current Issues:** PDF files passed directly to AI (unsupported), incorrect agent.generate format, inconsistent status updates, missing error handling.

### `src/mastra/tools/pdf-text-extractor.ts`
**Action:** Create  
**Impact:** Enables text extraction from PDFs for AI processing.

#### After
```ts
import { createTool } from '@mastra/core/tools';
import { z } from 'zod';
import pdfParse from 'pdf-parse';

export const pdfTextExtractorTool = createTool({
  id: 'pdf-text-extractor',
  description: 'Extracts text content from a PDF buffer.',
  inputSchema: z.object({ buffer: z.instanceof(Buffer) }),
  execute: async (inputData) => {
    const data = await pdfParse(inputData.buffer);
    return { text: data.text };
  },
});
```

### `src/mastra/workflows/summarize-estimate.ts`
**Action:** Modify  
**Impact:** Fixes API usage, adds text extraction, uses tools consistently.

#### Before
```ts
const fetchStep = createStep({
  id: 'fetch-file',
  inputSchema: z.object({ estimateRequestId: z.string(), fileUrl: z.string().url() }),
  outputSchema: z.object({ estimateRequestId: z.string(), base64: z.string() }),
  execute: async ({ inputData }) => {
    const res = await fetch(inputData.fileUrl);
    if (!res.ok) throw new Error(`Fetch failed: ${res.statusText}`);
    const buffer = Buffer.from(await res.arrayBuffer());
    return { estimateRequestId: inputData.estimateRequestId, base64: buffer.toString('base64') };
  }
});

const summarizeStep = createStep({
  id: 'summarize-file',
  inputSchema: z.object({ estimateRequestId: z.string(), base64: z.string() }),
  outputSchema: z.object({ estimateRequestId: z.string(), summary: z.string() }),
  execute: async ({ inputData, mastra }) => {
    const agent = mastra.getAgent('estimate-summarizer');

    const result = await agent!.generate([
      { type: 'text', text: 'Provide a structured technical summary of repairs and costs from this report.' },
      { type: 'file', data: inputData.base64, mimeType: 'application/pdf' },
    ]);

    return { estimateRequestId: inputData.estimateRequestId, summary: result.text };
  }
});
```

#### After
```ts
const fetchStep = createStep({
  id: 'fetch-file',
  inputSchema: z.object({ estimateRequestId: z.string(), fileUrl: z.string().url() }),
  outputSchema: z.object({ estimateRequestId: z.string(), buffer: z.instanceof(Buffer) }),
  execute: async ({ inputData }) => {
    const res = await fetch(inputData.fileUrl);
    if (!res.ok) throw new Error(`Fetch failed: ${res.statusText}`);
    const buffer = Buffer.from(await res.arrayBuffer());
    return { estimateRequestId: inputData.estimateRequestId, buffer };
  }
});

const extractStep = createStep({
  id: 'extract-text',
  inputSchema: z.object({ estimateRequestId: z.string(), buffer: z.instanceof(Buffer) }),
  outputSchema: z.object({ estimateRequestId: z.string(), text: z.string() }),
  execute: async ({ inputData, mastra }) => {
    const tool = mastra.getTool('pdf-text-extractor');
    const result = await tool!.execute({ buffer: inputData.buffer });
    return { estimateRequestId: inputData.estimateRequestId, text: result.text };
  }
});

const summarizeStep = createStep({
  id: 'summarize-text',
  inputSchema: z.object({ estimateRequestId: z.string(), text: z.string() }),
  outputSchema: z.object({ estimateRequestId: z.string(), summary: z.string() }),
  execute: async ({ inputData, mastra }) => {
    const agent = mastra.getAgent('estimate-summarizer');

    const result = await agent!.generate([
      {
        role: 'user',
        content: [
          { type: 'text', text: 'Provide a structured technical summary of repairs and costs from this report.' },
          { type: 'text', text: inputData.text }
        ]
      }
    ]);

    return { estimateRequestId: inputData.estimateRequestId, summary: result.text };
  }
});
```
